A [Nx](https://nx.dev/) Plugin that contains executors and generators to support scanning projects using [Sonar](https://www.sonarsource.com/products/sonarqube/)

## Installation

<Tabs>
<TabItem label="npm">

```sh
npm install @koliveira15/nx-sonarqube --save-dev
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm add -D @koliveira15/nx-sonarqube
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn add @koliveira15/nx-sonarqube -D
```

</TabItem>
</Tabs>
